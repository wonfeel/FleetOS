FleetOS - документация

Индекс всего, что лежит в docs/. Английская краткая версия - в корневом README.md.

С чего начать

ФайлЗачем
quickstart.html4 шага: поднять мост, поставить на первый компьютер. Начните отсюда.
fleetos_guide.htmlПолный гайд: мост, установка, дашборд, API-ключ, fleetctl.py, разбор проблем.
hardening_guide.htmlАдмин/безопасность: read-only ключ, /health//metrics, бэкап флота, откат ядра, подпись Raytower, свежие правки эмуляции.
guide.htmlОтдельный гайд именно по триангуляции вышек (Raytower): калибровка, raytower.lua.
