Bridge address

This page is a static file and can be hosted anywhere (e.g. GitHub Pages), but it always talks to bridge_server.py running on your own PC. Default is right if you're viewing this on the same PC running the bridge. Browsers block plain-http requests from an https page to anything other than 127.0.0.1/localhost, so a remote address (e.g. a Radmin IP) only works if you open this file locally (or allow insecure content for this site in your browser).

Terminal (master computer)

Runs on the master computer as if typed at its real shell prompt (via shell.run). A program that never exits (like re-running fleetos) will freeze this terminal until it's stopped some other way - same as it would in-game.
